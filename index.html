PyCode Conference 2020

PyCode Conference is a conference devoted to the Python language. The organizers are people from the language community.

The aim of the conference is to raise qualifications of participants and highlight the latest trends in Python. It is also important for us to spread good programming practices.

The conference is addressed to all Python programmers. It opens up to the community at every level of experience which means it is intended for advanced specialists as well as for beginners, including students.

Indranil Ghosh - Learn Lambda Calculus with Python

Speaker

Final year Masters student in Physics from Jadavpur University, Kolkata and also leading the R&D department of Sirpi Products and Services Private Limited. Coding in R/Python/Fortran. His core area of research is scientific computing.

Learn Lambda Calculus with Python

This is an introduction to the simple concepts of untyped Lambda calculus along with hands-on implementation of the Lambda function application in Python. We define a function and implement it with Python Lambda. We then discuss the concepts of "abstraction", "application" and "substitution". Finally, we study some combinators from combinatory logic, before ending with encoding datatypes in lambda calculus. This talk will be helpful to students with science and python backgrounds.

Paweł Stoworowicz - Python under the hood

Speaker

Software developer, electronics engineer, Python fan. Currently exploring area of serverless solutions and IoT. Guitar player in free time.

Python under the hood

There is no such thing as a free lunch - every abstraction brings some cost. While coding in Python we usually don't care about memory management, objects sizes, allocated resources and many other "low level" things. That's nice feature of language since we can focus on solving domain level problems. However, it is also worth being aware what's happening under Python's hood. Within this presentation I will focus on some details related to built-in containers, their implementation quirks and some interpreter level facts.

Hacking the Home Assignment

Home assignments are here to stay as they offer less biased views and are more focused on the quality of your work.

Unfortunately, if you’re a Junior Developer, the quality of work is usually what stands out. Your projects won’t look like what a professional reviewer is used to: They may be messy, half-working, or lacking documentation and instructions.

In this talk you'll learn an organized system for working on a small project while presenting best-practices. That means passing more interview stages, more options to choose from, and better bargaining power for your next job.
