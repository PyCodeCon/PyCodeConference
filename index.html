PyCode Conference 2021

PyCode Conference is a conference devoted to the Python language.
It is the meeting place for both experts and beginners.

The aim of the conference is to raise qualifications of participants and highlight the latest trends in Python. It is also important for us to spread good programming practices.

The conference is addressed to all Python programmers. It opens up to the community at every level of experience which means it is intended for advanced specialists as well as for beginners, including students.

Organizers

The organizers are people from the Python language community.

Main organizer

A graduate of the Koszalin University of Technology in the field of Computer Science. The owner of the Code Squat company where she works as an organizer and programmer.

Ajit Kumar

Organizer

Dr. Ajit Kumar is a Post-doctoral researcher in Soongsil University, Seoul, South Korea. He has completed his Ph.D. from the Department of Computer Science, Pondicherry University in 2018. His Ph.D. thesis titled "A Framework for Malware Detection with Static Features using Machine Learning Algorithms" focused on Malware detection using machine learning. He is working with Python since 2012 for his research work and other development work. He is also interested in web development, Information security, and Data Science. Python is his language of choice for all the programming related tasks. He has been motivating and training students to adopt Python as his programming language. He loves to write and share the article about Python and its applications.

Indranil Ghosh

Organizer

Ph.D. student in Applied Mathematics, Massey University, working on dynamical systems. His core area of research is scientific computing. Specializes in Condensed Matter Physics. Coding in R/Python/Fortran. Maintains an open source book about Mathematical Optimization using Python.
